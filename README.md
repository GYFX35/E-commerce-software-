# Global E-commerce and Dropshipping AI Assistant

This project is a comprehensive solution for global e-commerce and dropshipping, featuring a Chrome extension with an integrated AI assistant.

## Features

- **Chrome Extension**: Seamlessly integrates with major e-commerce platforms (Amazon, AliExpress, Shopify, etc.).
- **AI Assistant**: Provides product recommendations, market analysis, and automated descriptions.
- **Dropshipping Tools**: Product sourcing, price tracking, and inventory management.
- **Global Support**: Multi-currency and multi-language support.

## Project Structure

- `extension/`: Chrome extension source code.
- `webapp/`: Web application for managing dropshipping operations.
- `api/`: Backend API for AI and data processing.
